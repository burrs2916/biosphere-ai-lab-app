具体做法（不改你公理内容）
1️⃣ World 内部引入“强制检查点”

例如（示意）：

impl BasicWorld {
    fn commit_state_change(&mut self, change: WorldChange) -> Result<(), WorldAxiomViolation> {
        WorldAxioms::assert_time_irreversible(self.tick, change.next_tick)?;
        WorldAxioms::assert_causality(change.cause_tick, change.effect_tick)?;
        WorldAxioms::assert_existence_uniqueness(...)?;
        // etc.

        self.apply(change);
        Ok(())
    }
}


任何状态变更都必须经过这里。

2️⃣ 禁止绕过式 mutation（非常重要）

World 内部：

不暴露 pub fn set_tick(...)

不暴露 pub fn insert_relation_unchecked(...)

不允许外部直接修改核心状态结构

👉 World 是存在论的唯一入口。
